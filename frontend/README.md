# mentalAidPal Frontend (Vite + React)

This is a simple Vite + React frontend for the mentalAidPal demo. It provides pages for Home, Login, Register, Student Dashboard, Chancellor Dashboard and Admin Dashboard. The app is intentionally simple and uses localStorage to simulate registration and login for demo purposes.

Quick start (Windows PowerShell):

```powershell
cd 'C:\Users\Reginald\Documents\mentalaidpal_last\frontend'
npm install --legacy-peer-deps
npm run dev
```

The backend API endpoints are not implemented yet. We'll add FastAPI backend code next and wire the API to MySQL when you're ready.
